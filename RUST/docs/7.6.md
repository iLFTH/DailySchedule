+ 类型安全：  
    + 类型安全的语言可以避免类型间的无效计算
    + 类型安全的语言还可以保证内存安全
    + 类型安全的语言也可以避免语义上的逻辑错误
    + Rust 是一强类型且类型安全的静态语言。 Rust 中一切皆表达式 表达式皆有值，值皆有类型，所以可以说，Rust 中一切皆类型
